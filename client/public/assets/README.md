# PUDA Logo Assets

This directory contains the logo assets for Punjab Urban Development Authority (PUDA).

## Required Logo File

Please place the PUDA logo image file here with the following name:
- **File name:** `puda-logo.png`
- **Recommended format:** PNG with transparent background
- **Recommended size:** 512x512 pixels or higher (will be scaled down as needed)

## File Location

The logo file should be placed at:
```
client/public/assets/puda-logo.png
```

This file will be accessible at the URL: `/assets/puda-logo.png`

## Usage

The logo is used in the following locations:
1. **Login Page** - Main logo display
2. **App Layout Sidebar** - Logo in the sidebar header
3. **App Layout Header** - Logo in the main header (when sidebar is closed)

## Fallback

If the logo file is not found, the application will automatically fall back to a default icon. However, it's recommended to add the actual PUDA logo for proper branding.
